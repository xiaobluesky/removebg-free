## 🚀 部署说明

### 最低配置要求
- CPU: 2 核心
- RAM: 4GB (推荐 8GB)
- Python: 3.11+
- 存储：10GB

### Docker 部署 (推荐)
```bash
git clone https://github.com/xiaobluesky/removebg-free.git
cd removebg-free
docker-compose up -d
# 访问 http://localhost:3000
```

### 本地部署
```bash
# 后端
cd backend
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000

# 前端 (新开终端)
cd frontend
python -m http.server 3000
```
